# Birthday Surprise Site

Single-page countdown site built for your wife. Assets live alongside the HTML so it's easy to host anywhere or push to GitHub Pages.

## Files
- `index.html` — main page with live counters and gallery.
- `photo_2025-12-01_21-47-39.jpg` — hero background.
- `photo_2025-12-01_21-47-41.jpg`, `photo_2025-12-01_22-14-10.jpg` — gallery images.

## Quick start
Open `index.html` directly in a browser, or serve the folder with any static host.
